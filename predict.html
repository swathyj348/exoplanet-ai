Technical Architecture

Feature Engineering

Automated processing of 22+ astronomical parameters including orbital period, planet radius, stellar mass, and equilibrium temperature with log transformations for skewed distributions.

  • Orbital period normalization
  • Stellar parameter scaling
  • Missing value imputation
  • Outlier detection and handling

XGBoost Classifier

Gradient boosted decision trees optimized for NASA K2 data characteristics. Hyperparameter tuning with cross-validation ensures robust performance across diverse stellar populations.

  • 500 estimators with early stopping
  • Learning rate optimization
  • Feature importance ranking
  • Class imbalance handling

Bias Correction

Statistical bias correction applied to probability outputs to account for class distribution differences between training and real-world deployment scenarios.

  • Validation-based bias estimation
  • Per-class probability adjustment
  • Calibrated confidence scores
  • Uncertainty quantification
